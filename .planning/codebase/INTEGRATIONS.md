## Environment Configuration

**Required env vars (critical):**
- `APP_KEY` - Laravel app encryption key (generated by `artisan key:generate`)
- `APP_DEBUG` - Debug mode (boolean)
- `DB_CONNECTION` - Database type (`sqlite` or `mysql`)
- `NATIVEPHP_APP_ID` - Mobile app identifier (e.g., `com.example.app`)

**Optional env vars (NativePHP mobile):**
- `NATIVEPHP_APP_VERSION` - App version (e.g., `1.0.0`)
- `NATIVEPHP_APP_VERSION_CODE` - Android version code (integer, increments per release)
- `NATIVEPHP_DEVELOPMENT_TEAM` - iOS code signing team ID
- `NATIVEPHP_ANDROID_SDK_LOCATION` - Android SDK path
- `NATIVEPHP_GRADLE_PATH` - Gradle JDK path
- `NATIVEPHP_ANDROID_COMPILE_SDK` - Android compile SDK (default: 36)
- `NATIVEPHP_ANDROID_MIN_SDK` - Android minimum SDK (default: 33)
- `NATIVEPHP_ANDROID_TARGET_SDK` - Android target SDK (default: 36)

**Optional env vars (Email, unused):**
- `POSTMARK_API_KEY` - Postmark email service
- `RESEND_API_KEY` - Resend email service
- `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_DEFAULT_REGION` - AWS SES

**Optional env vars (Slack, unused):**

- `SLACK_BOT_USER_DEFAULT_CHANNEL` - Slack channel

**Secrets location:**

- `.env.android` - Android build-specific settings
